# Music Popularity & Audio Feature Analysis (Tableau)

## Project Overview

This project uses Tableau to analyze music popularity trends and audio features across genres and time. The objective is to demonstrate data visualization, exploratory analysis, and the ability to translate statistical patterns into business-relevant insights.

The analysis is framed from the perspective of music producers or executives using data to understand what drives commercial success.

## What This Project Shows

* Interactive Tableau dashboards analyzing genre popularity
* Use of reference lines and comparisons to identify above-average performers
* Trend analysis of music production and popularity over multiple decades
* Visual exploration of audio features such as energy, danceability, valence, and speechiness

## Key Insights

* Hip Hop/Rap, Pop, and Latin genres consistently rank above average in popularity
* Popular music has become more energetic and danceable over time
* Audio features related to rhythm and intensity appear more influential than musical key
